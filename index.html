JavaScript Topics Summary

1. Alerts

Alerts are used to display a message box to the user. The alert() function creates a popup box with a specified message and an OK button.

alert("Hello, World!");

2. Variables for Strings

String variables are used to store text. Strings are enclosed in either single quotes ('') or double quotes ("").

var greeting = "Hello, World!";

3. Variables for Numbers

Number variables are used to store numeric values. JavaScript supports both integers and floating-point numbers.

var age = 25;
var price = 19.99;

4. Variable Names Legal and Illegal

Variable names can include letters, digits, underscores, and dollar signs, but they must begin with a letter, underscore, or dollar sign. Reserved words cannot be used as variable names.

Legal: var name, _name, $name, name1;

Illegal: var 1name, name!, var, function;

5. Math Expressions: Familiar Operators

JavaScript supports basic arithmetic operations: addition (+), subtraction (-), multiplication (*), and division (/).

var sum = 5 + 3;
var difference = 5 - 3;
var product = 5 * 3;
var quotient = 5 / 3;

6. Math Expressions: Unfamiliar Operators

In addition to familiar operators, JavaScript includes modulo (%) for remainder, increment (++), and decrement (--).

var remainder = 5 % 3;
var increment = 5;
increment++;
var decrement = 5;
decrement--;

7. Math Expressions: Eliminating Ambiguity

Use parentheses to control the order of operations and eliminate ambiguity in mathematical expressions. 

var result = (5 + 3) * 2; // 16
var result = 5 + (3 * 2); // 11

8. Concatenating Text Strings

Use the + operator to concatenate strings.

var greeting = "Hello, " + "World!";

9. Prompts

The prompt() function displays a dialog box that prompts the user for input. The input can then be stored in a variable.

var name = prompt("Enter your name:");
alert("Hello, " + name + "!");
